Abrir CMD
Escribir cd y arrastrar la carpeta SASS
npm install -g sass (Se instala una vez por capeta)
sass --watch style.scss style.css (depende el nombre de cada archivo que utilicemos)
sass --watch carpeta/style.scss carpeta/style.css (si estan en carpeta)
